# skillswap-mvp

React Native (Expo) prototype for the SkillSwap assignment.

## How to run
1. Install Node.js and Expo CLI:

```bash
npm install -g expo-cli
```

2. Create or open the project and replace `App.js` with the provided `App.js`.

3. Run the app:

```bash
npm install
expo start
```

Open in Expo Go or an emulator.

## Files included
- A1/SRS.md (or SRS.docx if available)
- A1/UML/*.puml (PlantUML sources)
- App.js
- README.md
- run_commands.txt (git commands to create repo and push)
